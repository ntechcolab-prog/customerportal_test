# NETZSCH Customer Portal

Recriação do NETZSCH Customer Portal em HTML/CSS puro, tela por tela a partir do Figma.

## Estrutura

```
customerportal_test/
├── pages/        # Uma tela por arquivo HTML
├── assets/       # Imagens e outros recursos
├── README.md
└── CLAUDE.md
```

## Stack

- HTML + CSS puro (sem frameworks)
- Um arquivo por tela dentro de `/pages`
- Assets de imagem em `/assets`

## Telas

| Arquivo | Tela | Status |
|---|---|---|
| `pages/home.html` | Landing page (Home) | ✅ |
| `pages/login.html` | Tela de login | ⬜ |
| `pages/register.html` | Tela de cadastro | ⬜ |
| `pages/dashboard.html` | Dashboard principal | ⬜ |
